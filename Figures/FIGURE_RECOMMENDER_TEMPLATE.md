# Figure Recommender Template

## 新用户先看

如果你还没有运行过这些 notebook，可以先做最小准备：

```bash
pip install numpy pandas matplotlib seaborn jupyter
jupyter lab
```

然后按这个顺序使用：

1. 如果你还不知道该选什么图，先用这份文档做“图种推荐”。
2. 如果你已经拿到推荐结果，回到 [README_Figure.md](README_Figure.md) 查看 `图种选择表`、`风格规则表` 和 `Notebook 索引表`。
3. 打开对应的 `.ipynb` 文件，先替换数据，再微调样式。

如果你已经很明确自己要画哪一类图，可以直接去 [README_Figure.md](README_Figure.md) 找参考 notebook。

这个文档是仓库内的**推荐模板**，只负责两件事：

1. 推荐 1 到 3 个合适的图种
2. 推荐对应的参考文件

它**不负责**：

- 直接修改 notebook
- 教你完整绘图语法
- 替你做论文排版

## 使用目标

当你拿不准“该画什么图”时，用这个模板描述你的需求，然后输出：

- 推荐图种
- 推荐 notebook 文件
- 推荐理由
- 不推荐的图种

## 输入模板

请尽量补齐下面信息：

```text
1. 你的数据是什么？
   例如：基因表达、细胞比例、模型指标、时间序列、相关性矩阵、网络关系

2. 你最想表达的结论是什么？
   例如：组间差异、变化趋势、相关性、层级结构、组成占比、综合排名

3. 你的变量结构是什么？
   例如：单变量、双变量、三变量、多维指标、矩阵、网络、层级分类

4. 有没有这些结构？
   - 分组
   - 时间/梯度
   - 层级
   - 网络关系
   - 排名

5. 你更偏向哪种用途？
   - 论文主图
   - 补充材料
   - 汇报/PPT
   - 探索性分析

6. 你希望图更偏哪一类？
   - 稳妥易读
   - 信息密度高
   - 视觉风格强
```

## 推荐规则

### 1. 先按表达任务选图，不按“哪个图更炫”选图

- 组间差异优先：`对比点图.ipynb`、`云雨图.ipynb`
- 趋势优先：`折线图.ipynb`、`多变量变化趋势图.ipynb`
- 分布优先：`云雨图.ipynb`、`边际直方图.ipynb`
- 相关性/矩阵优先：`热力图.ipynb`
- 网络关系优先：`相关性网络热图.ipynb`、`相关性网络热图_蝴蝶图.ipynb`
- 层级结构优先：`旭日图.ipynb`、`嵌套饼图.ipynb`
- 综合排序优先：`综合排序气泡图.ipynb`、`雷达图.ipynb`

### 2. 新用户默认优先稳妥图种

如果没有明确理由，不优先推荐这些高复杂度图：

- `弦图.ipynb`
- `环形蜂窝条形图.ipynb`
- `相关性网络热图_蝴蝶图.ipynb`
- `旭日图.ipynb`
- `径向柱状图.ipynb`

### 3. 论文主图优先可读性

论文主图默认优先：

- `对比点图.ipynb`
- `折线图.ipynb`
- `热力图.ipynb`
- `云雨图.ipynb`
- `堆叠柱状图.ipynb`

### 4. 只有在确实需要时才推荐风格型图

以下图更适合“增强视觉风格”而不是“作为默认首选”：

- `玫瑰图.ipynb`
- `径向柱状图.ipynb`
- `弦图.ipynb`
- `环形蜂窝条形图.ipynb`

## 一个示例

### 输入

```text
数据是 4 个细胞亚群在 3 个处理组里的比例变化。
我最想表达的是组间组成差异。
有分组，没有时间轴，没有层级。
主要想放论文主图，优先稳妥易读。
```

### 输出

```text
推荐图种 1：
- 图种：堆叠柱状图
- 参考文件：堆叠柱状图.ipynb
- 适合原因：最直接表达各组组成占比变化，读者理解成本最低。

推荐图种 2：
- 图种：对比点图
- 参考文件：对比点图.ipynb
- 适合原因：如果你想突出某一个亚群在不同组之间的差异，对比点图更聚焦。

不推荐：
- 图种：旭日图
- 原因：当前没有明确层级结构，用层级图会增加理解负担。
```
