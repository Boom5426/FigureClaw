# Awesome Scientific Figures

这个目录不是一套完整教程，而是一个**科研绘图模板入口**。
新用户建议按下面顺序使用：

1. 先看 `图种选择表`，确认你要表达的任务类型。
2. 再看 `风格规则表`，避免把图画成“信息很多但不好读”的版本。
3. 最后到 `Notebook 索引表` 找具体参考文件。

如果你只想快速得到建议，请直接使用 [FIGURE_RECOMMENDER_TEMPLATE.md](FIGURE_RECOMMENDER_TEMPLATE.md)。

## 运行方式

安装基础依赖：

```bash
pip install numpy pandas matplotlib seaborn jupyter
```

启动 Notebook：

```bash
jupyter lab
```

## 风格规则表

| 图种类别 | 适合场景 | 不推荐场景 | 风格规则 | 常见误区 |
|---|---|---|---|---|
| 对比类 | 比较组间大小差异 | 类别太多、需要层级结构时 | 保持排序一致，颜色少而稳 | 同时使用太多颜色和标记 |
| 分布类 | 展示离散、集中、偏态 | 只想表达均值时 | 让原始分布可见，减少不必要装饰 | 只给均值却不展示分布 |
| 趋势类 | 时间、浓度、伪时间变化 | 横轴不是连续变量时 | 横轴顺序必须真实，突出主线 | 线太多、图例太乱 |
| 相关性/矩阵类 | 看整体模式、聚类关系 | 只比较少数几个变量时 | 优先控制配色和色阶范围 | 色盘过艳、没有统一尺度 |
| 网络类 | 强调连接关系和模块 | 节点很多且没有重点时 | 只保留关键节点和关键边 | 把所有边都画上去 |
| 层级类 | 展示父子结构和组成 | 没有清晰层级时 | 层级不要过深，标签要克制 | 用复杂层级图表达简单问题 |
| 气泡类 | 同时编码位置、大小、分组 | 点太密集时 | 气泡大小映射要有明确含义 | 把气泡大小当装饰而不是变量 |
| 环形类 | 强调周期、方向或视觉风格 | 需要精确比较长度时 | 优先少类别、强对比、简标签 | 为了“好看”牺牲可读性 |
| 多维综合类 | 少量维度的综合比较 | 维度太多、样本太多时 | 限制维度数，保持统一量纲 | 雷达图塞太多指标 |
| 论文主图通用规则 | 需要直接进论文或报告 | 只是探索阶段草图时 | 标题短、图例少、主结论突出、色彩统一 | 一个图里同时讲太多事情 |

## 使用建议

- 新用户优先从 `对比点图`、`折线图`、`热力图`、`云雨图` 开始。
- 如果你不确定图种，先使用 [FIGURE_RECOMMENDER_TEMPLATE.md](FIGURE_RECOMMENDER_TEMPLATE.md)。
- 真正替换数据时，优先找 notebook 里的数据读取或示例数据块，再改绘图参数，不要一开始就大改样式。
